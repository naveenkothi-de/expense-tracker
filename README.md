
A simple and interactive  Streamlit web app  to record and analyze your daily expenses.  
You can add, edit, delete, and visualize your spending with clean charts and summaries.
Features  
- Add, edit, and delete expenses  
- Filter by date or category  
- View monthly summaries  
- See total and category-wise spending  
- Interactive charts (bar and pie)  
- Data stored in a local CSV file (`expenses.csv`)
Tools & Technologies  
- **Python**  
- **Streamlit**  
- **Pandas**  
- **Matplotlib / Plotly**  
- **CSV file for storage**
Output
	•	Total spending per category
	•	Monthly expense trends
	•	Pie chart for category distribution
	•	Filtered expense table
Future Enhancements
	•	Add user login
	•	Connect to database (SQLite or Cloud)
	•	Budget alerts and savings insights
	•	Deploy on Streamlit Cloud or Azure
